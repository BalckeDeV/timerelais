# timerelais
--- 

We want to create a time relay system using a Raspberry Pi. According to ChatGPT, this should be feasible.
When a person presses a button, a signal is sent to the Raspberry Pi. A light controller provides information about the outdoor brightness.
If the ambient light level is low and someone presses the button, the indoor and outdoor lights of the life-saving house are activated.
During the night, the microcontroller dims the lights. In the bedroom, the lighting is adjusted to a calming tone (a very soft shade of blue).
