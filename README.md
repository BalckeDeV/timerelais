# timerelais
--- 
We want to create a timerelais with a raspberry pi. [ChatGPT](https://chatgpt.com/share/69e219a2-0b94-8395-90c9-154ad47d8c77) means that it will work.

If a person pushes a button than there should be a signal which goes to the raspberry pi. There is a light controller which gives an information about the lightness of the outside the house. 
If the lightness is low and the somebody push the button than the light inside and outside of the life saving house will be activated.
